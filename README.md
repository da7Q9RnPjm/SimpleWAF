## Features
- Real time report using Telegram Bot API
- Protect Range IP Deny
- Protect From IP Spam
- Protect Request URL
- Protect Request Server
- Protect Santy
- Protect Bots
- Protect Request Method
- Protect DoS, Union SQL, Click Jacking, Cookies, and Upload
